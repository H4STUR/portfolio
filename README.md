# System XD

A Windows XP-themed portfolio website built with React. Experience the nostalgia of the classic desktop environment while exploring my projects, skills, and background.

**Live Demo:** [portfolio.agares.co.uk](https://portfolio.agares.co.uk/)

## Features

### Interactive Desktop Environment

Fully functional Windows XP simulation with draggable icons, resizable windows, and a working taskbar. Browse through folders, open files, and discover what's on this "computer".

### Built-in Applications

| App | Description |
|-----|-------------|

## Tech Stack

**Frontend:** React 18 , Vite, react-rnd, react-dnd, Phaser

**Backend:** Laravel 11, PHP 8.2, SQLite

## Quick Start

```bash
# Frontend
npm install && npm run dev

# Backend (optional - for scoreboard & contact form)
cd backend
composer install
cp .env.example .env
php artisan key:generate && php artisan migrate
php artisan serve
```
